📋 SETUP (5 MINUTES)
═══════════════════════════════════════════════════════════════════════

1. Get Bot Token:
   • Message @BotFather on Telegram
   • Send: /newbot
   • Copy your bot token

2. Create Channel:
   • Public channel: @AAUCrushConnect
   • Add bot as admin
   • Grant posting permissions

3. Configure .env:
   BOT_TOKEN=your_token_here
   CHANNEL_ID=@AAUCrushConnect
   ADMIN_GROUP_ID=your_user_id

4. Install & Run:
   pip install -r requirements.txt
   python bot.py

5. Set Admin:
   • Message your bot: /start
   • Complete profile
   • Send: /set_admin YOUR_USER_ID

🛠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Bot not responding?
  → Check BOT_TOKEN
  → Verify bot is running
  → Check logs: bot.log

Channel not posting?
  → Bot is admin?
  → Posting permissions?
  → CHANNEL_ID correct?

Database errors?
  → Supabase credentials
  → Internet connection
  → RLS policies enabled

Verify setup:
  python verify_setup.py
